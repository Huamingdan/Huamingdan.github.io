3.4.3 图像标签：

标签的属性src中写的是一个路径

路径有两种写法：

       1、相对路径  相当于当前文件的路径

                  ./表示当前的路径

           ../当前路径的上一级目录

       2、绝对路径  绝对位置 不会随着当前文件的路径的位置改变而改变

Img标签的常用属性：

     src:指定图片路径

         alt:当图片找不到时候的提示文字

         width:指定图片的宽度，使用像素点 px单位，使用百分比指的是占据父元素的百分比

         height:指定图片的高度

         title：当鼠标移动到图片上的提示文字

3.4.9 div，span标签（重点）：

    在2008年之前，我们网页布局都是通过table来进行的。到了2008年自后，网页布局基本上都是通                                            过div+css来进行布局的。

    div表示一块区域。我们可以把整个网站划分成若干块内容，然后实现各自块的内容。div里面基本上什么标签都可以放。

Span也是代表一块小区域，只不过这个区域里面只能够放文本、图片、链接、表单元素等标签。

3.4.10 表单标签：

表单标签的作用：用来收集用户数据的，可以做注册登录等。

    表单的组成：表单域(form)  

                表单控件(input  textarea   select) 

                提示信息  label

    Input标签的分类：

        1、   Text          普通输入框

​ 2、 Password 密码输入框

​ 3、 Submit 提交按钮，会提交表单

​ 4、 Checkbox 多选框

​ 5、 radio 单选框

​ 6、 file 文件选择域

​ 7、 reset 重置按钮

​ 8、 image 图片，会提交表单，同时会将鼠标所点击的位置也提交到服务器

​ 9、 button 普通按钮，不会提交表单(如果非要用button提交表单得写js代码)

​ 10、hidden 隐藏域不是用来显示页面的，是用来向服务器提交隐藏字段的信息

​ form表单的属性：

​ 1.action 表单所提交地址

​ 2.method 请求方式 ：

​ get 会将请求参数拼接在url后面

​ post 会将请求参数放在请求体里面

3.4.11 其他标签：

​ Iframe：内联标签

​ Pre：照原样输出

块级元素：独占一行、可以设置宽高。div,form,table,p,pre,h1~h6,dl,ol,ul

行内元素：在一行显示、不可以设置宽高。span,a,label,select

行内块级元素：在一行显示、可以设置宽高。image input
